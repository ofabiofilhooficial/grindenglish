## Key Visual Elements

### Icons Used
- 📚 Lesson-level resources
- 📖 Grammar chapters / Stage resources
- 🗣️ Vocabulary entries
- ✓ Understood/completed
- ▼ Collapsed (click to expand)
- ▲ Expanded (click to collapse)
- 🎯 Learning goal
- [X] Remove/unlink

### Color Coding
- Primary blue: Grammar resources
- Accent color: Vocabulary resources
- Success green: Understood/completed items
- Muted gray: Collapsed/inactive items
- Border highlight: Currently expanded item

### Layout Principles
1. **Progressive Disclosure**: Resources collapsed by default
2. **Contextual Placement**: Stage resources appear with stage content
3. **Clear Hierarchy**: Stage resources → Lesson resources
4. **Visual Feedback**: Checkmarks for understood items
5. **Compact Design**: Minimal space when collapsed
6. **Scannable**: Icons and badges for quick identification
